## Bilibili2RSS

将B站的番剧更新转制为RSS输出

## 项目地址

https://github.com/wdssmq/Bilibili2RSS

## 使用方法（不想写UI和提交表单啊。。。）

https://bilibili2rss.bid/anime/5800 （例子）

以番剧妹抖龙为例，该番剧地址为

`https://bangumi.bilibili.com/anime/5800`

然后将地址修改为上述例子中的

`https://bilibili2rss.bid/anime/5800`

将改地址输入到你的RSS阅读器即可
